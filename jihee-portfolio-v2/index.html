프로세스
문제 정의 → 컨셉 → 시각화 → 결과물. 과정 중심의 포트폴리오.
의뢰 문의

문제 정의

사용자/클라이언트 요구, 레퍼런스 취합, 페르소나/톤앤무드 정리.

컨셉

컬러 팔레트(#fcffde 하이라이트), 얼굴/몸 비율, 헤어 텍스처, 루즈핏 실루엣 가이드.

시각화

러프 → 클린업(검은 외곽선 없음) → 주름/광택 → PNG 투명 산출.

결과물

  • A4 PNG(300dpi) 투명 / JPG 배경 포함 버전
  • 색상/텍스트 교정 1–2회 포함(프로젝트별 상이)
  • 상업 이용 라이선스 협의
